## 環境設定

本專案使用 Python 3，並需要安裝以下依賴庫來運行視覺化模組：

1.  **安裝 Python 依賴庫：**
    ```bash
    pip install pandas matplotlib
    ```
2.  **資料檔案：** 費用資料將儲存於 `expenses.csv` 檔案中。

## 執行

請按照以下步驟運行輸入與視覺化模組。

### 步驟 1: 執行輸入模組 (Input Module)

使用 `input.py` 來新增費用記錄。

```bash
python input.py
```
### 步驟 2: 生成圓餅圖
使用 `Visualization.py` 生成圓餅圖
```bash
python Visualization.py
```
